# 2.1 X-Tray 安装说明

## 2.1.1 产品组件清单

- X-Tray智能托盘主体 (后称X-Tray)
- 供电线

## 2.1.2 准备工作

1. 确认安装环境:请确保 **X-Tray智能托盘** 将被安置于干燥、清洁、无尘的环境中。

2. 检查供电线和所有附件是否完好无损。

3. 确认柜体内托盘的尺寸和位置,以确保X-Tray能够稳固地放置。

## 2.1.3 安装步骤

1. **放置X-Tray:** 将X-Tray轻轻放置于柜体内预定的托板上。确保托盘平稳无倾斜,以保证后续使用的精确性。

2. **检查水平仪:** 观察X-Tray正面的水平仪,确保X-Tray放置平稳。如水平仪指示不平衡,可适当调整X-Tray的位置,直到水平仪显示水平。

3. **接通电源:** 转至X-Tray的后视图,找到电源接线端子。使用随附的供电线,将其一端连接到X-Tray的电源接线端子,另一端连接到符合安全标准的电源插座。X-Tray可以使用一根公对公DC连接线从相邻的已供电的X-Tray上接电。注意，以后种方式供电时，相互连接的X-Tray最多不应超过4台。 

4. **安装脚垫:** 检查X-Tray底部的四个称重单元脚垫,确保它们紧密接触地面,以确保稳定性和准确性。

5. **开启电源:** 返回至X-Tray的正视图,找到电源开关,开启电源。检查设备是否正常启动,观察指示灯或屏幕是否按预期显示。

## 2.1.4 使用检查

1. 再次确认X-Tray的水平状态，开关应处于关闭状态，X-Tray上没有承载任何重物及标签。

2. 检查电源线和设备是否有任何损坏，确保所有连接都牢固无误。

3. 接电后，打开X-Tray的电源开关，听到内部蜂鸣器的滴鸣，则代表开机成功。 

## 2.1.5 注意事项

1. 如要挪动X-Tray，请务必在关闭开关及电源后进行。断电时，请先关闭X-Tray电源，再拔除电源线。

2. 保证X-Tray的通风良好，不要堵塞通风口。

3. 定期清理设备，避免灰尘积累影响性能和精度。

4. 若X-Tray出现故障，请勿自行拆修，应联系专业人员进行检修。
