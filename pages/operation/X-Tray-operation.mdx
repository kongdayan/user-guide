# 3.1 X-Tray 智能托盘操作指南

本操作指南旨在提供X-Tray智能托盘的基本操作步骤,包括电源的开关、网络连接以及调平校准。

## 3.1.1 开关电源

### 开启电源
1. 确保电源线已正确连接至X-Tray智能托盘和电源插座。
2. 找到X-Tray的电源开关,通常位于设备的侧面或前面。
3. 将电源开关置于“开”(ON)位置。此时,您应该能够看到指示灯亮起,表示电源已经开启。
4. 等待设备启动,此过程可能需要几秒钟至几分钟不等。

### 关闭电源
1. 在关闭电源前,请确保所有的操作都已经保存并退出。
2. 将电源开关置于“关”(OFF)位置,指示灯应熄灭,表示电源已经关闭。
3. 如不再使用,可从电源插座中断开电源线,以确保设备完全断电。

## 3.1.2 网络连接


1. **无线连接（如设备支持）：**
   - 工程师会默认帮助你通过串口工具写入无线连接信息
   - 设备通电启动后会自动进入网络连接模式
   - 在路由器的设备界面和EMQX的客户端界面可以看到已连接网络的设备

## 3.1.3 调平校准

1. 将X-Tray智能托盘放置在一个平稳的表面上。
2. 检查设备侧面或顶部是否有水平仪指示器。
3. 如果设备不平衡,调整设备下方的脚垫,直至水平仪指示器显示设备已平稳。
4. 在设备的用户界面中,找到校准选项（如果提供的话）,按照指示执行校准过程。这将确保在使用过程中测量和操作的准确性。

请在进行任何重要操作前确保设备已正确连接到网络并完成调平校准,以保障使用过程中的数据准确性和设备安全。
